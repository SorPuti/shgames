# Jogo de Cavernas - Repositório de Jogos em Bash

Este é um repositório contendo dois jogos simples criados com bash: `snake.sh` para o jogo da cobra (Snake) e `car.sh` para uma versão simplificada do carro 
(Car). Ambos os jogos foram escritos em Bash Script, que é executado diretamente no terminal no Linux.

## Como Jogar

### Instalação
1. Clone este repositório ou copie o código fonte diretamente para sua pasta de desenvolvimento.
2. Certifique-se de ter um terminal suportando bash (Bash Shell).

### Comandos de Execução:

**Para Snake:**
```
./snake.sh [Tamanho da cobra] [Intervalo entre movimentos]
```

**Para Car:**
```
./car.sh
```

### Exemplos de Jogos

1. Para iniciar o jogo da cobra com uma cobra que cresce até 8 células e intervalo de 0,5 segundos:
```
./snake.sh 8 0.5
```

2. Para iniciar a versão simples do carro:
```
./car.sh
```

### Como Jogar:

- Pressionando as teclas (w) para cima, (a) para esquerda, (s) para baixo e (d) para direita, você controlará o movimento da cobra.
- Use a letra "q" para sair do jogo.
